Die Programmiersprachen

Processing

Processing ist eine auf Java basierende Programmiersprache welche am 9. August
2001 von Casey Reas und Ben Fry veröffentlicht wurde. Viele Universitäten benutzen
Processing da die Sprache viele Anwendungsmöglichkeiten hat, von der Spieleentwicklung
über das Erstellen von Datenbanken bis zu Animationen. Ausserdem kann Processing für
Interaktionen mit diversen Hardware-Plattformen wie zum Beispiel Arduino genutzt werden.
Von Processing gibt es neben der Standartversion für den PC auch iProcessing welche
es möglich macht Anwendungen für Iphones zu zu entwickeln sowie auch Mobile Processing
welche das ganze auch für andere Mobilgeräte welche Java unterstützen unterstützt.

HTML

HTML steht für Hypertext Markup Language und wurde am CERN in Genf entwickelt.
Die Programmiersprache entstand zur gleichen Zeit wie das Internet da man eine
einfach zu verstehende Textauszeichnungssprache benötigte. Am 3. November 1992
erschien die erste Version von HTML. Der Syntax der Sprache ist relativ einfach, jedes
Element besteht aus einem Anfangstag, in dem Attribute definiert und werden, sowie
einem Endtag dass das Element abschliesst. Jedes Tag beginnt mit einem <-Zeichen und
endet mit einem >-Zeichen. HTML ist aber nur für die Struktur eines Textes zuständig,
nicht für das Design. Durch die einzelnen Tags, zum Beispiel einem Header wird zwar
klargestellt dass es sich um einen Titel handelt, aber nicht in welcher Schriftgrösse,
Farbe oder Schriftart das ganze dargestellt wird, diese Darstellung ist dann vom Browser
abhängig wenn sie nicht durch CSS extra definiert wird

CSS

CSS steht für Cascading Style Sheets auf deutsch gestufte Gestaltungsbögen. Diese Sprache
gehört zusammen mit HTML und Javascript zu den Kernsprachen des Internets. CSS enstand 1994
mit CHSS (Cascading HTML Style Sheets) am CERN in Genf um HTML-Elemente zu formatieren und
individuell anzupassen. CHSS folgte CSS2 (CSS Level 2) im Jahre 1998 und die aktuelle Version
CSS3 (CSS Level 3) welche seit 2000 in Entwicklung ist. Um ein spezifisches HTML ELement in
CSS anzupassen wird es in HTML mit einem "div" Element und einem dazugehörigen "class" zusatz
einer bestimmten Klasse zugeordnet werden. Dieses wird dann mit verschiedenen CSS Attributen
individuell angepasst. Damit HTML und CSS miteinander kommunizieren können gibt es 3 verschiedene
Arten der Einbindungen von CSS in HTML:
1. Direkt im HTML Tag des betreffenden Elements (Inline CSS)
2. In einem externen Dokument, auch Stylesheet genannt (External CSS)
3. In einem Style Element direkt im HTML Dokument (Internal CSS)
