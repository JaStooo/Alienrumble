Coden der Website

HTML Struktur

Um die ganze Website zu strukturieren wird HTML verwendet. Hier ein paar spezielle Komponenten:

Das Menü

Um HTML Komponenten in CSS zu bearbeiten wird das ganze als
div, also eine einzelne Klasse festgelegt. Dies dient dazu
um die Komponenten im CSS File zu definieren. Man kann sich
dass ganze als eine Art Variable für ein Bild vorstellen.
Navbar in der obersten Zeile stellt damit den Namen für die
Variable dar.

Danach wurde mit einem Header eine neue Zeile geschaffen und
ein Logo eingefügt, dies geschieht im img class Bereich, der
class-Bereich definiert wieder eine Variable für die CSS be-
arbeitung, der src-Bereich fügt dann das Element ein.

Nun wurde mit den href's die dahinterstehenden Seiten mit den
zugehörigend HTML Dokumenten verlinkt. Zuletzt wurde noch eine
Trennlinie hinzugefügt, die Dicke defniert und Schwarz gefärbt.

Der Downloadbereich

Im Downloadbereich wurde der Downloadbutton als Bild eingefügt
und mit einer Verknüpfung zum Spiel versehen. Mit dem download-
attribut wurde dann festgelegt dass die Verknüpfung herunter-
geladen werden sollte. Dahinter wurde noch der Name der Datei
Nach dem Download mit Game festgelegt. Der Button wurde noch
Individuell mit der Grösse angepasst.
